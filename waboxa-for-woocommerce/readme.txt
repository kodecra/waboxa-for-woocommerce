Send customizable WhatsApp messages for WooCommerce orders directly from your admin panel.

== Description ==
Waboxa for WooCommerce, mağazanızdaki siparişler için hızlı ve özelleştirilebilir WhatsApp mesajları göndermenizi sağlar.  
“Alındı”, “Kargo” ve “Teslim” mesajlarını tek tıkla gönderebilir, mesaj metinlerini admin panelinden kolayca düzenleyebilirsiniz.  

**Öne çıkan özellikler:**
- WooCommerce sipariş ekranında WhatsApp butonları  
- Admin panelinden düzenlenebilir mesaj şablonları  
- {fullname}, {order_id}, {courier_title}, {tracking_url}, {review_link} değişken desteği  
- Mobil ve masaüstü WhatsApp yönlendirmesi  
- Hezarfen & HepsiJET uyumlu kargo takip entegrasyonu  
- Veri toplamaz, tamamen güvenli ve hafif

== Installation ==
1. ZIP dosyasını indir.  
2. WordPress panelinden **Eklentiler → Yeni Ekle → Eklenti Yükle** menüsüne git.  
3. ZIP dosyasını yükleyip **Etkinleştir** butonuna tıklayın.  
4. Yönetim panelinde **Waboxa Mesajları** menüsüne gidin ve mesaj şablonlarınızı düzenleyin.  

== Frequently Asked Questions ==
= Değişkenleri nasıl kullanırım? =
Mesajlarda aşağıdaki değişkenleri kullanabilirsiniz:  
- `{fullname}` → Müşteri adı  
- `{order_id}` → Sipariş numarası  
- `{courier_title}` → Kargo firması  
- `{tracking_url}` → Takip bağlantısı  
- `{review_link}` → Değerlendirme sayfası linki  

= Hangi WhatsApp versiyonunu kullanır? =
Mobil cihazlarda **api.whatsapp.com**, masaüstünde **web.whatsapp.com** yönlendirmesi yapar.

= Eklenti güvenli mi? =
Evet. Waboxa herhangi bir veriyi dışarıya iletmez veya üçüncü taraf servislerle paylaşmaz. Tüm işlemler yerel WordPress ortamında çalışır.
